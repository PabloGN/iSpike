Documentation

The best to start with iSpike is a recent paper LazdinsFidjelandGamez11_iSpike.pdf, which describes the motivation behind iSpike, integration with other software, spike conversion methods, and performance and accuracy measurements.

The first version of iSpike was created by Edgars Lazdins, whose project report contains further background information and more details about the software architecture.

A short manual with instructions for installing and using iSpike is available: ispike-manual-2.0.pdf

Full source code documentation is available here.
